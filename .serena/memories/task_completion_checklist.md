# タスク完了時のチェックリスト

## 1. Lint実行
```bash
npm run lint
```
- ESLintでコードの静的解析を実行
- エラーや警告がないことを確認

## 2. ビルド確認
```bash
npm run build
```
- 本番ビルドが成功することを確認
- TypeScriptの型エラーがないことを確認
- ビルドプロセスでエラーが発生しないことを確認

## 3. 開発サーバーでの動作確認
```bash
npm run dev
```
- 開発サーバーを起動して実際の動作を確認
- 変更した機能が正しく動作することを確認
- UIの表示崩れがないことを確認

## 注意事項
- **テストは未設定**: このプロジェクトにはテストフレームワークが設定されていません
- **フォーマッターは未設定**: 自動フォーマッティングは現在利用できません
- **型チェック**: ビルドコマンドで型チェックが実行されます（`tsc --noEmit`相当）

## アーキテクチャ遵守の確認
### 依存関係のルール
- 内側の層（Domain）が外側の層（Infrastructure, Interface）に依存していないか確認
- 依存性注入（DI）を通じて依存関係が管理されているか確認

### ファイル配置の確認
- 新しいファイルが適切な層に配置されているか確認:
  - ビジネスロジック → `src/app/domain/`
  - データアクセス → `src/app/infrastructure/`
  - UI/Controller → `src/app/interface/`
