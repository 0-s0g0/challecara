# Challecara プロジェクト - 推奨コマンド

## 開発サーバー
```bash
npm run dev
```
開発サーバーを起動します（通常は http://localhost:3000）

## ビルド
```bash
npm run build
```
本番用にアプリケーションをビルドします

## 本番サーバー起動
```bash
npm start
```
ビルド後の本番アプリケーションを起動します

## Lint
```bash
npm run lint
```
ESLintでコードの静的解析を実行します

## パッケージインストール
```bash
npm install
```
依存パッケージをインストールします

## システムコマンド（macOS Darwin）
プロジェクトはmacOS（Darwin）で開発されています。基本的なUnixコマンドが使用可能です:
- `git` - バージョン管理
- `ls` - ディレクトリ一覧
- `cat` - ファイル内容表示
- `grep` - テキスト検索
- `find` - ファイル検索

## 注意事項
- このプロジェクトにはテストコマンドは未設定です
- フォーマッターは未設定です（ESLintのみ）
- データベース関連のコマンドは未設定です（Prismaは依存関係にありません）
